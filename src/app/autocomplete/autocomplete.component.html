The autocomplete is a normal text input enhanced by a panel of suggested options. Simple autocomplete Start by creating the autocomplete panel and the options displayed inside it. Each option should be defined by a mat-option tag. Set each option's value property to whatever you'd like the value of the text input to be when that option is selected.
